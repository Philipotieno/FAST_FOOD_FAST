How to test for the API endpoints on postman

  1. To Register a User follow these steps(POST Request)

    1. Copy the above URL on this page and to it add /api/v1/user/signup

    2. Paste it on postman params: Example app_name.herokuapp.com/api/v1/user/signup

    3. Go to the body and add user registration details in JSON format

      4. Example:

      { "username":"herokuapp", "email":"heroukuapp@gmaill.com", "password":"heroku1234" }

    4. Click send and wait for the response,incase of any errors check on the datails used to register, make sure to pu a comma after evey detail except the last details

  2. To Log a User follow these steps(POST Request)

    1. Copy the above URL on this page and to it add /api/v1/user/login'

    2. Paste it on postman params: Example app_name.herokuapp.com/api/v1/user/login'

    3. Go to the body and add user login details in JSON format

      4. Example:

      { "username":"herokuapp","password":"heroku1234" }

    4. Click send and wait for the response,incase of any errors check on the datails used, make sure to pu a comma after evey detail except the last details

  3. To Place an order follow this steps(POST Request)

    1. Copy the above URL on these page and to it add /api/v1/user/orders'

    2. Paste it on postman params: Example app_name.herokuapp.com/api/v1/user/orders'

    3. Go to the body and add order details in JSON format

      4. Example:

      { "food":"nyama","price":"1234" }

    4. Go to the Auth Header and add bearer token details from the log in response

    5. Click send and wait for the response,incase of any errors check on the datails used, make sure to put a comma after evey detail except the last details, make sure you have use the tokens to place an order

  4. To View all orders follow these steps(GET Request)

    1. Copy the above URL on this page and to it add /api/v1/user/orders'

    2. Paste it on postman params: Example app_name.herokuapp.com/api/v1/user/orders'

    3. Click send and wait for the response,incase of any errors check on the datails used, make sure to put a comma after evey detail except the last details, make sure you have use the tokens to place an order

  5. To View specific orders follow this steps(GET Request)

    1. Copy the above URL on this page and to it add /api/v1/user/orders/1'

    2. Paste it on postman params: Example app_name.herokuapp.com/api/v1/user/orders/1'

      1 represents the specific order_id

    3. Click send and wait for the response,incase of any errors check on the datails used, make sure to put a comma after evey detail except the last details, make sure you have use the tokens to place an order

  6. To Update a specific order follow these steps(POST Request)

    1. Copy the above URL on this page and to it add /api/v1/user/orders/1'

    2. Paste it on postman params: Example app_name.herokuapp.com/api/v1/user/orders/1'

      { "food":"nyama","price":"1234" }

    5. Click send and wait for the response,incase of any errors check on the datails used, make sure to put a comma after evey detail except the last details, make sure you have use the tokens to place an order
